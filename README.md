# TuTuck Server

> [!NOTE]
> TuTuck is in very early stage. Report any bugs and issues via GitHub issues. If you know Go and like the project - feel free to contribute, we really need contributors

## What's TuTuck? 
A messenger in terminal. Simple. Secure. Works via ssh.

### Why does it exist?
Well, its a messenger that is NOT webview or electron, NOT written in rust, works fast, does the job.

## How do I try it?
First, you need a server. You can self host it or ask a friend. Download latest server binaries for your OS in [Releases](https://github.com/tutuck-org/server/releases/latest/)

Once you got a server to connect to, choose a client you like. You can try the [TUI client](https://github.com/tutuck-org/client-tui/) or find other. Sadly, there's not many clients for now (but you can change it! soon there will be a wiki)

Download client, connect to the server... Done! Explore
