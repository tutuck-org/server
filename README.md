# TuTuck Server
[![Go Report Card](https://goreportcard.com/badge/github.com/tutuck-org/server)](https://goreportcard.com/report/github.com/tutuck-org/server)

A messenger in therminal. Fast, simple, secure. Works via SSH.
No electron / webview slop. No rust. No bloat.

## Get started
#### 1. Get a server
You can:
   * Self host it
   * Ask a friend

Download latest server binaries:
  [github.com/tutuck-org/server/releases/latest](https://github.com/tutuck-org/server/releases/latest/)

#### 2. Pick a client
* TUI client

  [github.com/tutuck-org/client-tui](https://github.com/tutuck-org/client-tui/)

#### 3. Connect and enjoy!

> [!NOTE]
> TuTuck is in very early stage. Report any bugs and issues via GitHub issues. If you know Go and like the project - feel free to contribute, we really need contributors
